Mima's Fast Food - Food Delivery App - UI/UX Design Case Study

Client: Mima's Fast Food

Project: Food Delivery App (Concept Project, 2023)

Tools Used: Figma

Role: Ui/Ux Designer

Project Overview

This was a personal exploration project I started in 2023 to practice designing for mobile applications. The concept was to create a food delivery app that made browsing, ordering, and tracking food faster and more user-friendly compared to existing apps.

Goals

  • Explore UI/UX principles in the food delivery niche.
  • Practice creating clean, appetizing interfaces with strong typography and color choices.
  • Build an end-to-end user flow: from food discovery → ordering → checkout → delivery tracking.

Process

  • User Flow Mapping: outlined how a first-time user would discover, order, and track food.
  • Wireframes: sketched low-fidelity layouts for onboarding, home, and checkout screens.
  • UI Exploration: experimented with warm color palettes (reds, oranges, yellows) to spark appetite and trust.

Status

This project was an uncompleted concept that I paused in 2023 because I had to go for NYSC Camp but it allowed me to strengthen my design workflow, learn more about design systems, and practice building mobile app interfaces.
